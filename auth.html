Pengenalan CSS

CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk mengatur tampilan dokumen HTML.

Sejarah CSS:

  • Diusulkan oleh HÃ¥kon Wium Lie pada tahun 1994
  • Versi pertama: CSS1 (1996)
  • CSS3 adalah versi modular yang dirilis bertahap sejak 1999

Fungsi CSS:

  • Mengatur tata letak halaman web
  • Mengontrol warna, font, dan ukuran teks
  • Membuat tampilan responsif
  • Menambahkan animasi dan efek visual

Contoh Kode CSS Sederhana:

body {
  font-family: Arial, sans-serif;
  background-color: #f0f0f0;
}

h1 {
  color: #333;
  text-align: center;
}

Syntax Dasar CSS

CSS terdiri dari selector dan declaration block yang berisi properti dan nilai.

Struktur Dasar:

selector {
  property: value;
  property: value;
}

Contoh:

p {
  color: blue;
  font-size: 16px;
  margin: 10px 0;
}

Cara Menambahkan CSS ke HTML:

  • Inline CSS: Langsung di elemen HTML 
    <p style="color: red;">Teks merah</p>
  • Internal CSS: Di dalam tag <style> di head 
    <style>
  p { color: red; }
</style>
  • External CSS: File terpisah (.css) yang dihubungkan 
    <link rel="stylesheet" href="style.css">

Selector CSS

Selector digunakan untuk memilih elemen HTML yang akan diberi style.

Jenis-jenis Selector:

  • Element Selector: Memilih berdasarkan nama tag 
    p { color: blue; }
  • Class Selector: Memilih berdasarkan class (diawali .) 
    .warning { color: red; }
  • ID Selector: Memilih berdasarkan id (diawali #) 
    #header { background: #333; }
  • Attribute Selector: Memilih berdasarkan atribut 
    input[type="text"] { border: 1px solid #ccc; }
  • Pseudo-class: Memilih state khusus 
    a:hover { color: green; }

Kombinasi Selector:

/* Semua elemen p dengan class warning */
p.warning { color: orange; }

/* Elemen dengan class warning dan danger */
.warning.danger { background: red; }

/* Child selector */
ul > li { list-style: none; }

Box Model

Setiap elemen HTML adalah sebuah box yang terdiri dari content, padding, border, dan margin.

Komponen Box Model:

  • Content: Area yang berisi konten (teks, gambar, dll)
  • Padding: Ruang antara content dan border
  • Border: Garis yang mengelilingi padding dan content
  • Margin: Ruang di luar border, memisahkan elemen dari elemen lain

Contoh:

.box {
  width: 300px;
  padding: 20px;
  border: 5px solid black;
  margin: 10px;
  background-color: lightblue;
}

Box Sizing:

Secara default, width dan height hanya mengatur ukuran content. Gunakan box-sizing untuk mengubah:

/* Total width = width + padding + border */
.box { box-sizing: border-box; }

Layout CSS

CSS menyediakan beberapa teknik untuk mengatur tata letak elemen di halaman web.

Display Property:

  • display: block - Elemen mengambil seluruh lebar (div, p, h1-h6)
  • display: inline - Elemen mengalir dalam baris (span, a, strong)
  • display: inline-block - Gabungan block dan inline
  • display: none - Menyembunyikan elemen

Positioning:

  • position: static - Posisi normal (default)
  • position: relative - Relatif terhadap posisi normal
  • position: absolute - Relatif terhadap elemen parent terdekat yang positioned
  • position: fixed - Relatif terhadap viewport (tetap saat scroll)
  • position: sticky - Gabungan relative dan fixed

Float:

.image {
  float: left;
  margin-right: 15px;
}

Flexbox

Flexbox adalah model layout satu dimensi untuk mengatur distribusi ruang dan alignment.

Konsep Dasar:

  • Flex Container: Elemen parent dengan display: flex
  • Flex Items: Elemen anak langsung dari container
  • Main Axis: Sumbu utama (default horizontal)
  • Cross Axis: Sumbu tegak lurus main axis

Container Properties:

.container {
  display: flex;
  flex-direction: row | row-reverse | column | column-reverse;
  justify-content: flex-start | flex-end | center | space-between | space-around;
  align-items: stretch | flex-start | flex-end | center | baseline;
  flex-wrap: nowrap | wrap | wrap-reverse;
}

Item Properties:

.item {
  order: <integer>;
  flex-grow: <number>;
  flex-shrink: <number>;
  flex-basis: <length> | auto;
  align-self: auto | flex-start | flex-end | center | baseline | stretch;
}

CSS Grid

CSS Grid adalah sistem layout dua dimensi yang powerful untuk membuat desain kompleks.

Konsep Dasar:

  • Grid Container: Elemen dengan display: grid
  • Grid Item: Elemen anak langsung dari container
  • Grid Line: Garis pembatas kolom dan baris
  • Grid Track: Ruang antara dua garis (kolom atau baris)
  • Grid Cell: Perpotongan antara kolom dan baris
  • Grid Area: Satu atau lebih cell yang membentuk persegi panjang

Container Properties:

.container {
  display: grid;
  grid-template-columns: 100px auto 100px;
  grid-template-rows: 50px 1fr 50px;
  grid-gap: 10px;
  grid-template-areas: 
    "header header header"
    "sidebar main aside"
    "footer footer footer";
}

Item Properties:

.item {
  grid-column-start: 1;
  grid-column-end: 3;
  grid-row-start: 1;
  grid-row-end: 2;
  grid-area: header;
}

Responsive Design

Desain responsif membuat website menyesuaikan tampilan berdasarkan ukuran layar perangkat.

Viewport Meta Tag:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Media Queries:

/* Untuk layar lebar minimal 600px */
@media (min-width: 600px) {
  .container {
    width: 80%;
    margin: 0 auto;
  }
}

/* Untuk layar maksimal 480px */
@media (max-width: 480px) {
  nav {
    display: none;
  }
}

Unit Responsif:

  • % - Relatif terhadap parent
  • vw/vh - Relatif terhadap viewport width/height
  • rem - Relatif terhadap root font-size
  • em - Relatif terhadap font-size elemen saat ini

Transisi & Animasi

CSS memungkinkan pembuatan efek transisi dan animasi tanpa JavaScript.

Transisi (Transition):

.button {
  background: blue;
  transition: background 0.3s ease, transform 0.2s;
}

.button:hover {
  background: darkblue;
  transform: scale(1.05);
}

Animasi (Animation):

@keyframes slidein {
  from { transform: translateX(100%); }
  to { transform: translateX(0); }
}

.slide-element {
  animation: slidein 0.5s ease-in-out;
}

Property yang Dapat Dianimasikan:

  • transform (scale, rotate, translate, skew)
  • opacity
  • color, background-color
  • width, height
  • Dan banyak lainnya

Transform

CSS transform memungkinkan modifikasi ruang koordinat dari elemen.

Fungsi Transform:

  • translate() - Memindahkan elemen 
    transform: translate(50px, 100px);
  • rotate() - Memutar elemen 
    transform: rotate(45deg);
  • scale() - Mengubah ukuran elemen 
    transform: scale(1.5);
  • skew() - Miringkan elemen 
    transform: skew(30deg, 20deg);
  • matrix() - Gabungan semua transform

Transform Origin:

Mengatur titik pusat transformasi:

transform-origin: 50% 50%; /* Default */
transform-origin: left top;
transform-origin: 20px 50px;

CSS Variables

CSS Variables (Custom Properties) memungkinkan penyimpanan nilai yang dapat digunakan kembali.

Deklarasi Variable:

:root {
  --primary-color: #4285f4;
  --padding: 15px;
  --max-width: 1200px;
}

Penggunaan Variable:

.header {
  background-color: var(--primary-color);
  padding: var(--padding);
  max-width: var(--max-width);
}

Keuntungan:

  • Mudah dikelola dan diubah
  • Nilai dapat diubah dengan JavaScript
  • Scope global atau lokal
  • Fallback value

Preprocessor (Sass/Less)

CSS preprocessor adalah bahasa scripting yang memperluas CSS dan dikompilasi ke CSS.

Fitur Sass:

  • Variables: 
    $primary-color: #333;
body { color: $primary-color; }
  • Nesting: 
    nav {
  ul { margin: 0; }
  a { color: blue; }
}
  • Mixins: 
    @mixin border-radius($radius) {
  -webkit-border-radius: $radius;
     -moz-border-radius: $radius;
      -ms-border-radius: $radius;
          border-radius: $radius;
}

.box { @include border-radius(10px); }
  • Inheritance: 
    %message-shared {
  border: 1px solid #ccc;
  padding: 10px;
}

.success { @extend %message-shared; }

CSS Frameworks

Framework CSS menyediakan kode siap pakai untuk mempercepat pengembangan.

Popular CSS Frameworks:

  • Bootstrap - Framework paling populer dengan sistem grid dan komponen siap pakai
  • Tailwind CSS - Utility-first CSS framework
  • Foundation - Framework responsif yang powerful
  • Bulma - Framework berbasis Flexbox
  • Materialize - Berdasarkan Material Design Google

Contoh Penggunaan Bootstrap:

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.2/dist/css/bootstrap.min.css" rel="stylesheet">

<div class="container">
  <div class="row">
    <div class="col-md-4">Kolom 1</div>
    <div class="col-md-4">Kolom 2</div>
    <div class="col-md-4">Kolom 3</div>
  </div>
</div>

Best Practices

Berikut adalah praktik terbaik dalam menulis CSS:

Organisasi Kode:

  • Gunakan metodologi seperti BEM (Block, Element, Modifier)
  • Pisahkan CSS ke beberapa file berdasarkan komponen
  • Gunakan komentar untuk membagi bagian

Performance:

  • Minimalkan penggunaan selector kompleks
  • Hindari !important
  • Gunakan shorthand properties
  • Minify CSS untuk produksi

Maintainability:

  • Gunakan naming convention yang konsisten
  • Hindari inline style
  • Gunakan CSS variables untuk nilai yang sering digunakan

Contoh BEM:

/* Block */
.menu { ... }

/* Element */
.menu__item { ... }

/* Modifier */
.menu__item--active { ... }
