Pengenalan JavaScript

JavaScript adalah bahasa pemrograman tingkat tinggi yang digunakan untuk membuat halaman web interaktif.

Sejarah JavaScript:

  • Dibuat oleh Brendan Eich pada tahun 1995
  • Versi pertama dibuat dalam 10 hari
  • Nama awal: Mocha → LiveScript → JavaScript
  • ES6 (2015) membawa banyak fitur modern

Fungsi JavaScript:

  • Membuat halaman web interaktif
  • Memvalidasi form
  • Membuat animasi
  • Berinteraksi dengan API
  • Membangun aplikasi web kompleks

Contoh Kode JavaScript Sederhana:

// Menampilkan alert
alert('Hello World!');

// Mengubah konten HTML
document.getElementById('demo').innerHTML = 'Halo JavaScript!';

Syntax Dasar JavaScript

JavaScript memiliki aturan penulisan (syntax) yang harus diikuti.

Struktur Dasar:

  • Case Sensitive: JavaScript membedakan huruf besar dan kecil
  • Statement: Diakhiri dengan titik koma (;)
  • Komentar: 
    // Komentar satu baris
/* Komentar 
   multi baris */

Contoh Syntax:

// Deklarasi variabel
let nama = 'Budi';

// Fungsi
function sapa() {
  return 'Halo ' + nama;
}

// Pemanggilan fungsi
console.log(sapa());

Cara Menambahkan JavaScript ke HTML:

  • Internal: Dalam tag <script> 
    <script>
  alert('Hello!');
</script>
  • External: File terpisah (.js) yang dihubungkan 
    <script src="script.js"></script>

Variabel & Tipe Data

Variabel digunakan untuk menyimpan data dalam program.

Deklarasi Variabel:

  • var - Scope function (legacy)
  • let - Scope block (bisa diubah)
  • const - Scope block (konstan)

Contoh:

var nama = 'Budi'; // Legacy
let umur = 20;     // Bisa diubah
const PI = 3.14;   // Konstan

Tipe Data:

  • Primitif:
    • String - Teks ('Hello')
    • Number - Angka (123, 3.14)
    • Boolean - true/false
    • null - Nilai kosong
    • undefined - Belum didefinisikan
    • Symbol - Unik (ES6)
  • Non-Primitif:
    • Object - {nama: 'Budi'}
    • Array - [1, 2, 3]
    • Function - function() {}

Operator

Operator digunakan untuk melakukan operasi pada variabel dan nilai.

Jenis Operator:

  • Aritmatika: +, -, *, /, %, ** 
    let hasil = 10 + 5; // 15
  • Penugasan: =, +=, -=, *=, /= 
    let x = 10;
x += 5; // x = x + 5
  • Perbandingan: ==, ===, !=, !==, >, < 
    5 == '5'  // true (nilai sama)
5 === '5' // false (tipe berbeda)
  • Logika: &&, ||, ! 
    true && false // false
true || false // true
!true         // false

Kondisi & Percabangan

Percabangan digunakan untuk mengeksekusi kode berbeda berdasarkan kondisi.

If-Else:

let nilai = 85;

if (nilai >= 90) {
  console.log('A');
} else if (nilai >= 80) {
  console.log('B');
} else {
  console.log('C');
}

Switch:

let hari = 'Senin';

switch(hari) {
  case 'Senin':
    console.log('Hari kerja');
    break;
  case 'Sabtu':
    console.log('Weekend');
    break;
  default:
    console.log('Hari lainnya');
}

Ternary Operator:

let usia = 20;
let status = (usia >= 18) ? 'Dewasa' : 'Anak-anak';

Perulangan

Perulangan digunakan untuk mengeksekusi blok kode berulang kali.

For Loop:

for (let i = 0; i < 5; i++) {
  console.log(i); // 0, 1, 2, 3, 4
}

While Loop:

let i = 0;
while (i < 5) {
  console.log(i);
  i++;
}

Do-While Loop:

let i = 0;
do {
  console.log(i);
  i++;
} while (i < 5);

For-Of (Array):

let fruits = ['Apple', 'Banana', 'Orange'];
for (let fruit of fruits) {
  console.log(fruit);
}

Fungsi

Fungsi adalah blok kode yang dapat dipanggil berulang kali.

Function Declaration:

function sapa(nama) {
  return 'Halo ' + nama;
}
console.log(sapa('Budi')); // Halo Budi

Function Expression:

const sapa = function(nama) {
  return 'Halo ' + nama;
};

Arrow Function (ES6):

const sapa = (nama) => {
  return 'Halo ' + nama;
};

// Atau lebih singkat:
const sapa = nama => 'Halo ' + nama;

Parameter Default (ES6):

function sapa(nama = 'Tamu') {
  return 'Halo ' + nama;
}

Array

Array digunakan untuk menyimpan koleksi data.

Deklarasi Array:

let fruits = ['Apple', 'Banana', 'Orange'];

Operasi Array:

// Akses elemen
fruits[0]; // Apple

// Tambah elemen
fruits.push('Mango'); // di akhir
fruits.unshift('Grape'); // di awal

// Hapus elemen
fruits.pop(); // hapus akhir
fruits.shift(); // hapus awal

// Panjang array
fruits.length; // 3

Array Methods:

// Map
let numbers = [1, 2, 3];
let doubled = numbers.map(n => n * 2); // [2, 4, 6]

// Filter
let evens = numbers.filter(n => n % 2 === 0); // [2]

// Reduce
let sum = numbers.reduce((total, n) => total + n, 0); // 6

Object

Object digunakan untuk menyimpan data dalam bentuk key-value pairs.

Deklarasi Object:

let person = {
  name: 'Budi',
  age: 20,
  greet: function() {
    return 'Halo, saya ' + this.name;
  }
};

Akses Properti:

person.name; // Budi
person['age']; // 20
person.greet(); // Halo, saya Budi

Class (ES6):

class Person {
  constructor(name, age) {
    this.name = name;
    this.age = age;
  }
  
  greet() {
    return `Halo, saya ${this.name}`;
  }
}

let budi = new Person('Budi', 20);
budi.greet(); // Halo, saya Budi

DOM Manipulation

DOM (Document Object Model) adalah representasi HTML yang bisa dimanipulasi dengan JavaScript.

Seleksi Elemen:

// By ID
let header = document.getElementById('header');

// By Class
let items = document.getElementsByClassName('item');

// By Tag
let paragraphs = document.getElementsByTagName('p');

// Query Selector
let firstItem = document.querySelector('.item');
let allItems = document.querySelectorAll('.item');

Manipulasi Konten:

// Ubah konten
element.innerHTML = 'Konten baru';

// Tambah class
element.classList.add('active');

// Hapus class
element.classList.remove('active');

// Ganti atribut
element.setAttribute('src', 'new-image.jpg');

Buat Elemen Baru:

let newDiv = document.createElement('div');
newDiv.textContent = 'Div baru';
document.body.appendChild(newDiv);

Event Handling

Event adalah aksi yang terjadi di halaman web, seperti klik, hover, submit form, dll.

Event Listener:

let button = document.getElementById('myButton');

// Cara 1: Property
button.onclick = function() {
  alert('Button diklik!');
};

// Cara 2: addEventListener (lebih fleksibel)
button.addEventListener('click', function() {
  alert('Button diklik!');
});

Jenis Event Umum:

  • click - Ketika elemen diklik
  • mouseover/mouseout - Mouse masuk/keluar elemen
  • keydown/keyup - Ketika tombol keyboard ditekan/dilepas
  • submit - Ketika form disubmit
  • load - Ketika halaman selesai dimuat

Event Object:

button.addEventListener('click', function(event) {
  console.log(event.target); // Elemen yang diklik
  event.preventDefault(); // Mencegah aksi default
});

ES6+ Features

ES6 (ECMAScript 2015) memperkenalkan banyak fitur baru ke JavaScript.

Fitur ES6+ Penting:

  • Let & Const - Deklarasi variabel block-scoped
  • Arrow Functions - Sintaks fungsi lebih singkat
  • Template Literals - String dengan variabel 
    `Halo ${nama}`
  • Destructuring - Ekstrak nilai dari array/object 
    let [a, b] = [1, 2];
let {name, age} = person;
  • Spread/Rest Operator - ... 
    let arr = [...arr1, ...arr2];
function sum(...numbers) { }
  • Modules - Import/export kode antar file 
    import { fungsi } from './module.js';
export const nilai = 10;

Async Programming

JavaScript adalah bahasa single-threaded yang menggunakan konsep asynchronous untuk operasi non-blocking.

Callback:

function fetchData(callback) {
  setTimeout(() => {
    callback('Data diterima');
  }, 1000);
}

fetchData(data => {
  console.log(data); // Data diterima (setelah 1 detik)
});

Promise:

function fetchData() {
  return new Promise((resolve, reject) => {
    setTimeout(() => {
      resolve('Data diterima');
    }, 1000);
  });
}

fetchData()
  .then(data => console.log(data))
  .catch(err => console.error(err));

Async/Await:

async function getData() {
  try {
    let data = await fetchData();
    console.log(data);
  } catch (err) {
    console.error(err);
  }
}

Error Handling

Error handling digunakan untuk menangani kesalahan yang mungkin terjadi saat eksekusi kode.

Try-Catch:

try {
  // Kode yang mungkin error
  let hasil = x / y;
} catch (error) {
  // Tangani error
  console.error('Terjadi error:', error.message);
} finally {
  // Selalu dijalankan
  console.log('Selesai');
}

Throw Error:

function bagi(a, b) {
  if (b === 0) {
    throw new Error('Pembagian dengan nol');
  }
  return a / b;
}

Error Types:

  • Error - Generic error
  • SyntaxError - Kesalahan sintaks
  • ReferenceError - Referensi tidak valid
  • TypeError - Operasi pada tipe yang salah
