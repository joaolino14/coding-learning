Pengenalan HTML

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat halaman web.

Sejarah HTML:

  • Diciptakan oleh Tim Berners-Lee pada tahun 1991
  • Versi pertama: HTML 1.0 (1991)
  • HTML 5 (versi terbaru) dirilis tahun 2014

Fungsi HTML:

  • Membuat struktur halaman web
  • Menampilkan konten (teks, gambar, video)
  • Membuat form interaktif
  • Mengorganisir konten dengan elemen semantik

Contoh Kode HTML Sederhana:

<!DOCTYPE html>
            <html>
            <head>
              <title>Judul Halaman</title>
            </head>
            <body>
              <h1>Ini Heading</h1>
              <p>Ini paragraf.</p>
            </body>
              </html>

Struktur Dasar HTML

Dokumen HTML memiliki struktur dasar yang terdiri dari beberapa bagian utama:

1. Deklarasi DOCTYPE

Mendefinisikan versi HTML yang digunakan:

<!DOCTYPE html>

2. Elemen <html>

Elemen root yang membungkus seluruh konten HTML:

<html lang="en">
           <!-- Konten HTML -->
           </html>

3. Bagian <head>

Berisi metadata tentang dokumen:

<head>
            <meta charset="UTF-8">
            <title>Judul Halaman</title>
            </head>

4. Bagian <body>

Berisi konten yang ditampilkan di browser:

<body>
            <h1>Judul Konten</h1>
            <p>Paragraf konten...</p>
            </body>

Tag List (ul, ol, li)

List atau daftar digunakan untuk menampilkan item-item dalam urutan tertentu (ordered list) atau tidak berurutan (unordered list).

Unordered List (<ul>):

Menampilkan daftar dengan bullet (â€¢) tanpa urutan.

  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

Ordered List (<ol>):

Menampilkan daftar dengan angka atau huruf sebagai urutan.

<ol>
  <li>Buka Visual Studio Code</li>
  <li>Buat file HTML</li>
  <li>Tulis kode dasar HTML</li>
</ol>

Nested List (Daftar Bersarang):

Kamu bisa menaruh list di dalam list lain.

<ul>
  <li>Frontend
    <ul>
      <li>HTML</li>
      <li>CSS</li>
    </ul>
  </li>
  <li>Backend</li>
</ul>

<li> (List Item)

<li> adalah singkatan dari List Item. Tag ini digunakan untuk membuat satu item dalam sebuah daftar.

Tapi <li> tidak berdiri sendiri â€” dia harus berada di dalam:

  • <ul>
  • <ol>

Kenapa <li> Harus Di Dalam <ul>/<ol>?

Karena tag <li> hanya valid digunakan di dalam <ul>, <ol>, atau <menu>.

Jika tidak berada di dalam tag yang sesuai, maka:

  • Browser akan bingung konteksnya
  • Struktur HTML jadi tidak valid
  • Bisa menyebabkan tampilan kacau atau error di beberapa browser

Tag Tautan (a)

Tag <a> digunakan untuk membuat tautan atau hyperlink di halaman web. Tautan bisa mengarah ke:

  • Halaman lain di situs yang sama
  • Website eksternal
  • Bagian tertentu di halaman (anchor link)
  • File yang bisa diunduh

Struktur Dasar:

<a href="URL">Teks Tautan</a>

Contoh:

<a href="https://www.google.com"
>Kunjungi Google</a>

Tautan ke Halaman Lokal:

<a href="halaman2.html">Ke Halaman 2</a>

Tautan dengan Target:

Gunakan atribut target="_blank" untuk membuka link di tab baru.

<a href="https://youtube.com" target="_blank">Buka YouTube</a>

Tautan ke Bagian Halaman:

Gunakan #id untuk lompat ke elemen tertentu dalam halaman yang sama.

<a href="#footer">Lompat ke Footer</a>

Tautan untuk Unduh File:

<a href="ebook.pdf" download>Unduh Ebook</a>

Tag Gambar (img)

Tag <img> digunakan untuk menampilkan gambar pada halaman web. Tag ini termasuk **self-closing** (tidak membutuhkan tag penutup).

Struktur Dasar:

<img src="gambar.jpg" alt="Deskripsi Gambar">

Keterangan atribut:

  • src (source): URL atau path ke file gambar
  • alt (alternative text): Teks alternatif jika gambar gagal dimuat (juga penting untuk SEO dan aksesibilitas)

Contoh Gambar dari Folder Lokal:

<img src="images/logo.png" alt="Logo Website">

Contoh Gambar dari Link Internet:

<img src="https://www.example.com/ 
  foto.jpg" alt="Foto Online">

Menentukan Ukuran Gambar:

<img src="gambar.jpg" alt="Gambar" width="300" height="200">

Lebih baik atur ukuran gambar menggunakan CSS agar lebih fleksibel.

Membuat Gambar Jadi Tautan:

Tag Tabel (table)

Tag <table> digunakan untuk membuat tabel di halaman web. Tabel berguna untuk menampilkan data dalam bentuk baris dan kolom.

Struktur Dasar Tabel:

<table>
  <tr>
    <th>Nama</th>
    <th>Umur</th>
  </tr>
  <tr>
    <td>Andi</td>
    <td>20</td>
  </tr>
  <tr>
    <td>Budi</td>
    <td>22</td>
  </tr>
</table>

Keterangan Tag:

  • <table> : Membungkus seluruh isi tabel
  • <tr> (Table Row) : Membuat baris dalam tabel
  • <th> (Table Header) : Judul kolom (hurufnya tebal dan rata tengah secara default)
  • <td> (Table Data) : Isi sel dari tabel

Contoh Tampilan:

Nama Umur
Andi 20
Budi 22

Tips Tambahan:

  • Gunakan CSS untuk mempercantik tampilan tabel
  • Gunakan border, cellpadding, dan cellspacing untuk mengatur jarak dan garis
  • Gunakan tag <thead>,<tbody>, dan <tfoot> untuk struktur tabel yang lebih rapi

Tag Formulir (form)

Tag <form> digunakan untuk membuat formulir di halaman web, yaitu tempat pengguna bisa mengisi data seperti nama, email, dan lainnya. Form biasanya digunakan untuk login, pencarian, pendaftaran, dll.

Struktur Dasar Formulir:

<form action="/submit" method="post">
      
  <label for="nama">Nama:</label><br>
  <input type="text" id="nama" name="nama"><br>

  <label for="email">Email:</label><br>
  <input type="email" id="email" name="email"><br>

  <input type="submit" value="Kirim">
</form>

Penjelasan Atribut Penting:

  • action: URL tujuan saat form dikirim (misalnya ke server)
  • method: Cara pengiriman data, biasanya GET atau POST

Tag Input Umum di Dalam Form:

  • <input type="text"> : Input teks biasa
  • <input type="email"> : Input khusus email
  • <input type="password"> : Input password (tertutup)
  • <input type="radio"> dan <input type="checkbox"> : Pilihan
  • <textarea> : Kolom input banyak baris
  • <select> : Dropdown pilihan

Tag Layout div, span

Tag <div> dan <span> digunakan untuk pengelompokan dan pengaturan layout konten di HTML. Keduanya tidak memiliki tampilan visual secara default, tetapi sangat penting dalam struktur halaman web.

Tag <div>:

Digunakan untuk membungkus elemen blok seperti paragraf, gambar, form, tabel, dan lain-lain. 

<div>
  <h2>Judul Artikel</h2>
  <p>Isi artikel...</p>
</div>

Catatan: Tag <div> adalah elemen **block-level**, artinya akan memulai baris baru.

Tag <span>:

Digunakan untuk membungkus teks atau elemen kecil di dalam baris, biasanya untuk memberi gaya dengan CSS. 

<p>Ini adalah <span style="color: red;">teks berwarna merah</span> di dalam paragraf.</p>

Catatan: Tag <span> adalah elemen **inline**, artinya tidak membuat baris baru.

Perbedaan div vs span:

div span
Elemen blok (block-level) Elemen sebaris (inline)
Digunakan untuk struktur besar Digunakan untuk bagian kecil dalam teks
Membuat baris baru Tidak membuat baris baru

Tips:

  • Gunakan <div> untuk membagi halaman jadi bagian: header, konten, sidebar, footer, dll.
  • Gunakan <span> untuk menandai kata/kalimat tertentu untuk diberi gaya CSS
  • Selalu beri class atau id untuk styling dan manipulasi dengan CSS/JavaScript

Semantic HTML

Semantic HTML adalah penggunaan tag HTML yang memiliki makna jelas terhadap isi yang dibungkusnya. Tujuannya adalah agar struktur dokumen lebih terorganisir, mudah dibaca oleh developer, dan dipahami dengan baik oleh mesin pencari maupun teknologi bantu seperti screen reader.

Contoh Tag Semantik:

  • <header> â€“ Bagian atas halaman, biasanya berisi logo atau navigasi
  • <nav> â€“ Menandai bagian navigasi
  • <main> â€“ Menandai konten utama halaman
  • <article> â€“ Konten mandiri seperti artikel berita atau posting blog
  • <section> â€“ Bagian dari konten yang bertema sama
  • <aside> â€“ Konten sampingan (sidebar, info tambahan)
  • <footer> â€“ Bagian bawah halaman, biasanya berisi kontak, copyright, dsb.

Contoh Penggunaan:

<header>
  <h1>Belajar HTML</h1>
</header>

<main>
  <article>
    <h2>Apa itu Semantic HTML?</h2>
    <p>Tag semantik membuat struktur halaman lebih bermakna...</p>
  </article>
</main>

<footer>
  <p>© 2025 TeknoDev</p>
</footer>

Perbedaan dengan Tag Non-Semantik:

Tag Semantik Tag Non-Semantik
<header>, <main>, <footer> <div>, <span>
Memiliki makna jelas tentang isi Tidak menjelaskan arti dari konten
Lebih baik untuk SEO dan aksesibilitas Kurang deskriptif untuk bot/search engine

Tips:

  • Gunakan semantic tag untuk struktur halaman yang lebih baik
  • Gabungkan semantic tag dengan CSS agar tampil menarik
  • Sangat disarankan untuk keperluan SEO dan pengembangan profesional

Komentar di HTML

Komentar di HTML digunakan untuk menambahkan catatan atau penjelasan dalam kode yang tidak akan ditampilkan di browser. Ini sangat berguna untuk dokumentasi dan kolaborasi antar developer.

Format Komentar:

<!-- Ini adalah komentar -->

Contoh Penggunaan:

<!-- Bagian header website -->
<header>
      Selamat Datang di TeknoDev
    </header>

Catatan penting

  • Jangan gunakan komentar HTML untuk menyimpan informasi sensitif seperti password,karena tetap bisa dilihat oleh pengguna dengan melihat source code halaman

  • Komentar tidak mendukung nested(tidak bisa menaruh komentar didalam komentar)

    • Contoh Kode

    <!-- Ini benar -->

<!--
  <p>Paragraf ini tidak akan muncul</p>
-->

<!-- Ini salah -->
<!-- <p> <!-- nested komentar --> </p> -->

    Kapan Sebaiknya menggunakan Komentar?

    • Untuk menjelaskan bagian kode yang kompleks
    • Memberi tahu developer lain tentang tujuan kode tertentu
    • Mengomentari kode sementara saat debugging
    • Menandai TODO atau bagian yang perlu diperbaiki di masa depan
    • Saat ingin menandai bagian tertentu dari layout(header,footer,sidebar)

Character Entity (Simbol Khusus)

Dalam HTML, ada beberapa karakter yang memiliki arti khusus (seperti <, >, dan &). Jika kamu ingin menampilkan simbol tersebut sebagai teks biasa, kamu harus menggunakan HTML Character Entities.

Kenapa Perlu Character Entity?

  • Karena beberapa karakter punya fungsi khusus dalam HTML (misalnya < untuk membuka tag).
  • Untuk menampilkan karakter yang tidak ada di keyboard.

Contoh Umum:

Entity Hasil Deskripsi
&lt; < Kurang dari ( < )
&gt; > Lebih dari ( > )
&amp; & Simbol ampersand ( & )
&quot; " Tanda kutip ganda ( " )
&nbsp; [Â ] Spasi non-breaking
&copy; © Simbol copyright
&reg; ® Simbol registered trademark
&hearts; ♥ Simbol hati (â™¥)

Cara Penulisan:

Gunakan format &[nama]; atau &#[kode];. Contoh:

<p>2 &lt; 5 artinya 2 lebih kecil dari 5</p>
<p>&copy; 2025 TeknoDev</p>

Tips:

  • Gunakan referensi resmi jika butuh simbol lebih banyak.
  • Gunakan &nbsp; untuk membuat spasi tambahan.
  • Entity sangat penting untuk menjaga validitas kode saat menampilkan simbol khusus.
